Vous retrouverez toutes les informations néscessaire sur la page de votre application dans l'onglet "Applications d'entreprise". Le "ClientID" se trouve dans l'onglet "Vue d'ensemble". Le "Client secret" se trouve dans l'onglet "Certificats & secrets". Le "Discovery URI" peut être trouvé dans l'onglet "Points de terminaison" sur la page "Vue d'ensemble".
